# Paper Finalize

## 检查

- target journal format
- references
- figures and supplementary
- data/code availability
- cover letter
- author declarations
- `paper-verify` full mode result
- critical `review_findings`

## 阻塞

- full verify failure blocks finalization.
- critical review finding blocks finalization unless forced checkpoint approves continuing.

## 输出

- submission checklist
- export package notes
- final checkpoint request
