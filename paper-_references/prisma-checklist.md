# PRISMA Checklist Reference

Use this reference when `paper-research` is asked for a systematic review or meta-analysis.

- Define eligibility criteria before screening.
- Record databases, search strings, and search date.
- Track deduplication, screening, exclusion reasons, and included studies.
- Separate background citations from direct evidence.
- Preserve uncertainty and heterogeneity.
- Never upgrade a background citation into direct support without reading the relevant source.
